# Plan 07-01: Enrich Mock App with Deep Multi-Step Request Flows

## Objective
Transform the mock app services from shallow single-span endpoints into realistic multi-step processing chains with:
- Manual child spans for DB queries, cache lookups, validation
- Deeper call trees (8-12 spans per checkout trace)
- New endpoint variety for richer resource names in APM
- Fix port conflict (payment.js on 8082 instead of 8083)

## Wave: 1
