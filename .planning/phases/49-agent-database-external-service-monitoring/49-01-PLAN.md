# Plan 49-01: Node and Go Outbound Tracing

<objective>
Implement Outbound HTTP request tracing and Database driver query wrapping with basic PII Regex Obfuscation for the Node.js and Go APM agents.
</objective>

<requirements>
- Node.js must trace `http.request`, `https.request`, and global `fetch`.
- Node.js must hook `pg` and `mysql2` modules and capture SQL.
- Go must implement a `database/sql` transparent tracing wrapper (`telemetry.DB`).
- All captured SQL string and numerical literals MUST be scrubbed and replaced with `?` natively.
</requirements>

## Tasks

### 1. Implement Node.js HTTP/DB Hooks
<read_first>
- `agents/node/instrumentation.ts`
</read_first>
<action>
In `agents/node/instrumentation.ts`:
1. Use `Hook(['pg', 'mysql2'], ...)` to intercept database connections.
2. Proxy `Client.prototype.query` (pg) and `Connection.prototype.query`/`execute` (mysql2).
3. Create spans with `type: 'sql'`, set `span.resource` to the obfuscated query operation.
4. Provide a regex replacing SQL string limits and integers `/(['"]).*?\1|(\b\d+\b)/g` with `?`. Set `db.query` metadata with the scrubbed output.
5. Override `http.request`, `http.get`, `https.request`, `https.get` to inject `.setHeader('x-easymonitor-trace-id')` dynamically.
6. Wrap `global.fetch` identically.
</action>
<acceptance_criteria>
- Outbound REST requests carry parent context ids automatically.
- PG/MySQL operations dynamically build valid spans encapsulating query duration.
- PII limits effectively zero-touch sanitize identifiers.
</acceptance_criteria>

### 2. Implement Go SQL Wrapper
<read_first>
- `agents/go/telemetry.go`
</read_first>
<action>
Create a new file `agents/go/db.go` exposing:
1. `type DB struct { *sql.DB }` extending standard `database/sql`.
2. Provide `QueryContext`, `ExecContext`, and `QueryRowContext` methods.
3. For each wrapped execution, `telemetry.StartSpanFromContext` with `type: "sql"`.
4. Compile standard obfuscation regexp `regexp.MustCompile(` + "`" + `(['"]).*?\1|(\b\d+\b)` + "`" + `)`.
5. Measure timespan and terminate cleanly returning the unmolested underlying rows.
</action>
<acceptance_criteria>
- Clean syntactic sugar exists allowing standard Go services to map `db = telemetry.WrapDB(sql.Open(...))` instantly.
- Regex obfuscation guarantees safe span context extraction.
</acceptance_criteria>
