# Plan 44-02: Polyglot Agents Data Structures and Transport

## Objective
Implement the standardized Datadog v0.4 MessagePack payload Data Structures and HTTP POST transport mechanisms in all four primary agents (Go, Java, Node, Rust). This prepares each agent to buffer and flush `[][]DatadogSpan` payloads to the `node-agent` backend.
