# Plan 3: Admin User Management Panel

<objective>
Add admin-only API endpoints for user CRUD operations and a React admin page showing a users table with ability to add and remove users. Observer role users cannot see nor access the admin section.
</objective>

<must_haves>
- Admin-only API endpoints for user CRUD
- Admin guard middleware on backend

- Add user form (username, password, role)
- Delete user capability
- Admin section hidden from Observer users
</must_haves>

<action>
1. Create `dashboard/src/components/auth/AdminGuard.tsx`:
   ```tsx
   import { Navigate, Outlet } from 'react-router-dom';
   import { useAuth } from '../../contexts/AuthContext';

   export function AdminGuard() {
     const { user } = useAuth();
     if (user?.role !== 'Admin') {
       return <Navigate to="/" replace />;
     }
     return <Outlet />;
   }
   ```

2. Add admin API functions to `dashboard/src/lib/api.ts`:
   ```tsx
   export async function fetchUsers(): Promise<UserInfo[]> { ... }
   export async function createUserApi(username: string, password: string, role: string): Promise<UserInfo> { ... }
   ```
