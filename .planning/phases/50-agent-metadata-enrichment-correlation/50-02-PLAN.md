# Plan 50-02: Universal Log Correlation

<objective>
Inject internal `trace_id` and `span_id` mapping tokens directly into application execution logs (via global context scopes) to unify stdout text traces natively.
</objective>

<requirements>
- Node: Monkey-patch the internal prototype format of the standard `console` module cleanly.
- Java: Utilize dynamic reflection hooking SLF4J's `MDC` to pass contexts cleanly bypassing compile-time JVM requirements.
- Go: Implement idiomatic manual injection wrappers natively returning strings.
</requirements>
