# Phase 45: Agent Native Instrumentation Modules - Rust
**Objective:** Implement native tracing layers for `reqwest` HTTP clients to propagate context headers, alongside the existing `actix_web` server middleware.

## Context
Rust is currently using `tracing-subscriber` for span capture and an `actix_middleware` for inbound request injection. We need a `reqwest` integration to propagate span IDs downstream.
