# Plan 2: Service Map Dashboard — Interactive Topology Graph with Health Indicators

<objective>
Build a DataDog-style interactive service topology map using React Flow (or pure SVG/Canvas). Services are rendered as nodes with colored health rings, edges show call direction and traffic volume. Clicking a node shows a side panel with RED metrics. The map supports zoom, pan, and auto-layout.
</objective>

<read_first>
- dashboard/src/lib/api.ts
</read_first>
<action>
Add types and fetch function:

```typescript
export interface ServiceMapNode {
  service: string;
  total_requests: number;
  error_rate: number;
  avg_duration_ms: number;
  p95_duration_ms: number;
  status: 'healthy' | 'warning' | 'error';
  node_type: 'service' | 'database' | 'external' | 'cache';
}

export interface ServiceMapEdge {
  source: string;
  target: string;
  requests: number;
  error_rate: number;
  avg_duration_ms: number;
}

export interface ServiceMapResponse {
  nodes: ServiceMapNode[];
  edges: ServiceMapEdge[];
}

export async function fetchServiceMap(from?: string): Promise<ServiceMapResponse> {
  const params = from ? `?from=${from}` : '';
  const res = await apiFetch(`/api/v1/apm/service-map${params}`);
  if (!res.ok) throw new Error('Failed to fetch service map');
  return res.json();
}
```
</action>

Create `dashboard/src/components/service-map/ServiceMapGraph.tsx`:

This is the core visualization component using React Flow for the topology graph.

```typescript

  nodes: ServiceMapNode[];
  edges: ServiceMapEdge[];
  onNodeClick: (service: string) => void;
  selectedNode: string | null;
}
```

**Custom Node Component** (`ServiceNode`):
1. Circle/rounded-rect shape (64x64px)
2. Outer health ring: green="healthy", amber="warning", red="error"
3. Inner icon based on `node_type`:
   - "service" → Server icon
   - "database" → Database icon
   - "external" → Globe icon
   - "cache" → Layers/Zap icon
4. Service name label below
5. Request count badge (abbreviated: 1.2K, 3.4M)
6. Subtle glow effect on hover
7. Selected state: blue ring + scale up

**Edge Configuration:**
- Animated dashes for active traffic (based on request volume)
- Thickness proportional to log(requests)
- Color: gray default, red if error_rate > 10%
- Arrow markers at target end
- Label showing request count on hover

**Layout:**
- Use dagre auto-layout algorithm for clean DAG positioning (top-to-bottom or left-to-right)
- If dagre is not available, use a simple force-directed layout or manual grid positioning
- Support zoom, pan, fit-to-view

**Integration:**
```tsx
<ReactFlow
  nodes={flowNodes}
  edges={flowEdges}
  nodeTypes={{ serviceNode: ServiceNode }}
  fitView
  attributionPosition="bottom-left"
>
  <Controls />
  <Background variant="dots" gap={20} size={1} color="#1f2937" />
</ReactFlow>
```

Dark theme: background `#0a0a0f`, nodes `bg-gray-800`, edges `stroke-gray-600`.
